# 🎮 Tic Tac Toe Game

A modern and interactive **Tic Tac Toe Game** built using React, TypeScript, Tailwind CSS, and Vite.

This project features a clean UI, smooth gameplay experience, winner detection, draw detection, toast notifications, and local storage persistence.

# ✨ Features

- 🎯 Two-player turn system
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Reset game functionality
- 🔔 Toast notifications using React Hot Toast
- 💾 LocalStorage persistence
- ⚡ Fast development with Vite
- 🎨 Modern UI using Tailwind CSS
- 📱 Responsive design
- 🧩 Component-based architecture
- 🔷 Built with TypeScript

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

---

## 2️⃣ Enter Project Folder

```bash
cd tic-tac-toe-game
```

---

## 3️⃣ Install Dependencies

```bash
pnpm install
```

---

## 4️⃣ Start Development Server

```bash
pnpm dev
```

---

# 🏗️ Build For Production

```bash
pnpm build
```
